4.3.1 Register as A Service Provider

Â This function is for users who register as a service provider in the system

•         Navigation Process:

 

 

Iservz Home Page

 

 

•         To register as a service provider, a user clicks on the Service Provider Application on the Iservz Home Page

 

 

 

 

 

 

 

 

 

 

 

Service Provider Application Page

 

•         After filling in the application form, the user will be redirected to the application successful page and has to wait for the admin to request a meeting and accept their application.
