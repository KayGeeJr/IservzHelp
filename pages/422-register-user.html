4.2.2 Register User

This page contains the form a user has to fill when registering on a new system.

•         Navigation Process:

 

•         The user must click the Register Now button on the bottom of the Iservz home screen.

 

Register Page

 

•         After filling in all the information, the user clicks apply to successfully register on the system

 

 

 

 

Reference

Control

Functionality Description

1

Name Textbox

The name textbox is for users to enter their name which will be their default username

2

Surname Textbox

The surname textbox is for user to enter their surname

3

Gender Drop Down List

The gender dropdown list is for users to pick their gender which is already in the system’s database

4

Email Address Textbox

The email address textbox is for users to enter their email address which they will use on the system and will be the primary mode of communication between the user and the system

5

Age Number Control

The age number control is for users to pick their age. The minimum age a user can pick is 18

6

Cell Number Textbox

The cell number textbox is for user to enter their cell phone numbers. The cell phone number must be entered in a 123-456-7890 format so it can be easily verified

7

Street Number Textbox

The street number textbox is for users to enter the street number of their address.

8

Street Name Textbox

The street name textbox is for users to enter the street name of their address.

9

Province Drop Down List

The province dropdown list is for users to pick the province they reside in.

10

City Drop Down List

The city dropdown list is for users to enter the city they reside in. The cities will be populated based on which province the user picks.

11

Picture Upload Button

The picture upload button is for user to choose a profile picture which will be their avatar and provider will be able to identify them.

12

ID Number Textbox

The id number textbox is for the user to enter a valid id number on the system. If it is not a valid id number, the system will provide errors for the user to fix

13

Apply Button

The Apply button is when the user has filled in all their information and are ready to register on the system.
