4.6.1 Add Service Category

Â This function is for the admin to add a new service category into the system

•         Navigation Process:

 

Administrator Home Screen

•         To access the service categories, the user must click the service categories button

 

Service Categories Page

 

•         To add a new service, the admin must click the Add New Category button on the page

 

Add Service Category Page

 

•         After filling the form, the administrator must click the Add Service Category button

 

Reference

Controls

Functionality Description

1

Category Name Textbox

The category name textbox is for the admin to enter a name for the new service category

2

Category Description Textbox

The category description textbox is for the admin to enter a description of the new service category

3

Add Service Category Button

The add service category button submits the form and creates a new service category on the system
