Controls

Functionality Description

Login Button

The Login button is for users who already exist on the system and do not necessarily need to create a new Iservz account. Once clicked, the login button will redirect the user to the login page where they will fill in their credentials and login.

Service Provider Application

The Service Providers Application button is for service providers who want to offer their services to consumers on the system.  They will need to provide their details on the system and will be accepted by the system’s admin after doing a standard verification of the details.

View More Button

The View More button is for users to view the services under the service categories that are displayed on the screen. This is for the user to get more  information on the services performed by the system.

Register Now Button

The Register Now button is for new user on the system, more specifically, consumers. Once clicked the button will redirect the user to the user registration page where they will provide their information
