4.5.1 Accept Service Provider Application

Â This function is for the admin to accept/reject a service provider application

•         Navigation Process:

 

 

Administrator Home Page

 

 

•         To accept/reject a service provider application, the administrator clicks on the SP Applications button on the navbar of the home screen which will direct them to the SP Application Screen

 

 

 

 

 

 

 

 

 

 

SP Application Screen

•         Accept Service Provider

•         To accept the service provider application, the administrator clicks on the Approve button which will send an email to the service provider that their application has been approved.

 

Reject Service Provider

 

•         To reject a service provider application, the admin clicks on the Decline button which will send an email to the service provider that their application has been rejected.
